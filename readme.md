# Terrible AI Agent

by N.D. Jia

A browser extension to have an "AI" agent browse the web for you.

> <img width="651" height="528" alt="Image" src="https://github.com/user-attachments/assets/cb576b9a-6fdb-4126-8c76-c510f53bc119" />

## How it works

Type a prompt in the text field and click Submit.

The extension then proceeds to ignore your input and click a random link on the currently-active page.

## How to install

- [Clone or download repository here](https://github.com/NDJia/terrible-browser-agent)
- **Firefox**: [install a temporary extension](https://extensionworkshop.com/documentation/develop/temporary-installation-in-firefox/)
- **Chromium browsers**: [test a browser extension locally](https://dev.to/cassidoo/how-to-test-a-browser-extension-locally-4bne)
- **Other browsers**: your guess is as good as mine tbh
